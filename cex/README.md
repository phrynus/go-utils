### 一、基础必备配置

- `getServerTime`：获取交易所服务器时间（订单签名、时间戳验证核心）

### 二、核心市场信息

- `initExchangeInfo`：初始化合约交易对基础信息（含合约类型、杠杆范围、最小下单量等，下单前必须确认）

### 三、关键行情数据

- `getOrderBook`：获取指定合约订单簿（买一卖一及深度，下单定价参考）
- `getTicker`：获取单个合约实时行情（最新价、涨跌幅、资金费率等核心指标）
- `getMarkPrice`：获取指定合约标记价格（合约爆仓/平仓计算的核心价格，必用）
- `getOHLCV`：获取指定合约K线数据（技术分析基础，如15分钟/1小时线）
- `getFundingRate`：获取当前资金费率（永续合约持仓费用计算依据）

### 四、核心订单操作

- `createOrder`：创建合约订单（支持开仓/平仓、多/空、限价/市价，最核心功能）
- `cancelOrder`：取消指定未成交订单（订单调整必备）
- `cancelAllOrders`：取消指定合约所有未成交订单（紧急情况快速清挂单）

### 五、关键查询功能

- `getOrder`：查询单个订单详情（状态、成交情况、保证金等）
- `getOpenOrders`：查询所有未成交订单（跟踪挂单状态）
- `getPositions`：查询所有合约持仓（持仓数量、均价、盈亏、强平价等，核心风控）
- `getBalance`：查询合约账户余额（可用保证金、总权益等，资金管理核心）

### 六、杠杆与保证金基础设置

- `setLeverage`：设置合约杠杆倍数（影响盈亏放大比例，必调参数）
- `setMarginMode`：设置保证金模式（交叉/逐仓，资金风险控制核心）
- `addMargin`：为持仓增加保证金（降低爆仓风险，实用功能）

### 七、实时数据监听（高频交易必备）

- `watchOrderBook`：实时监听合约订单簿更新（盘口动态实时获取）
